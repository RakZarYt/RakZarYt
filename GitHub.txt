cls

// Список файлов и папок

| ИНИЦИАЛИЗАЦИЯ ПРОЕКТА |

1) Переходим в папку проекта, с помощью cd
2) Инициализируем репозиторий

// Инициализация проекта
git init




| Изменения проекта |

// Статистика проекта
git status

// Добавления файла к слежке
git add <file/.>
* file - Файл или папка за которой будем следить
* Вместо <file>, пишем . - что-бы выбрать всё!

// Перестоём следить за файлом или папкой
git rm --cached <file>
* file - Файл или папка за которым не будем следить
